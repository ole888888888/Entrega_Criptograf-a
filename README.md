Se trata de una entrega para la asignatura de Algebra lineal y estructuras matemáticas de la UGR.  
El archivo def.cpp contiene el código para encriptar un mensaje ascii, y el archivo num.cpp únicamente encripta enteros.  
Se hace uso de la librería GMP que permite usar enteros de precisión arbitraria se puede encontrar la descarga en [gmplib.org](https://gmplib.org/)
